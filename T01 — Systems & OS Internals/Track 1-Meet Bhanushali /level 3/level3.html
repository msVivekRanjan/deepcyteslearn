Memory Isolation Constraints

Address Space Layout Randomization (ASLR): Shuffles code, stack, libraries, and heap base addresses to make memory offset prediction impossible.
Data Execution Prevention (DEP / NX): Marks stack/heap memory sections writeable or executable but NEVER both, stopping stack payload launches.
