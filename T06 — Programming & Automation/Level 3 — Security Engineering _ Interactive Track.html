Module 01

Security Scripts

Small programs that do repetitive security jobs — like checking for open doors automatically.

A security script is code (Python, Bash, PowerShell) that performs security tasks: monitoring, detection, response, reporting. Think of it as a tireless digital helper.

TypePurpose
ReconnaissanceGather info (port scan, WHOIS)
Vulnerability scannerFind weak points (CVE checker)
Log analysisSpot suspicious activity
Incident responseBlock IP after N failures
Password auditingTest password strength in audits
Network monitoringWatch traffic for anomalies

Ethical reminder

Only run security scripts on systems you have written permission to test. Unauthorized scanning is illegal.

Module 03

Log Parsing

Reading machine 'diary entries' to spot trouble — automatically.

Every system writes logs constantly. The skill is filtering noise to find the few lines that matter, using patterns (regex) and counters.

Log typeWhy it matters
AuthenticationDetect brute-force logins
FirewallSpot intrusion attempts
Web serverCatch SQLi / XSS attempts
DNSFind C2 malware traffic
SIEM aggregatedCorrelate everything in one place
# detect brute force
import re
from collections import defaultdict
PATTERN = re.compile(r'Failed password.*from (\d+\.\d+\.\d+\.\d+)')
fails = defaultdict(int)
for line in open('/var/log/auth.log'):
    m = PATTERN.search(line)
    if m: fails[m.group(1)] += 1
for ip, n in fails.items():
    if n >= 5: print(f'[ALERT] {ip} -> {n} failed attempts')

Module 06

CLI Development

Command-line tools are how engineers compose pipelines and automate everything.

Good CLIs are predictable: clear subcommands, well-named flags, helpful errors, and machine-readable output.

# argparse skeleton
import argparse
parser = argparse.ArgumentParser(
    prog='sectools',
    description='Security Engineering Toolkit',
    epilog='Example: sectools scan -t 192.168.1.1 -p 1-1024')
sub = parser.add_subparsers(dest='cmd')
scan = sub.add_parser('scan')
scan.add_argument('-t', '--target', required=True)
scan.add_argument('-p', '--ports', default='1-1024')
scan.add_argument('--json', action='store_true')
