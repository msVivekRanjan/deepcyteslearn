01

Logic

The foundation of thinking in code

Logic is how computers make decisions. Everything boils down to True or False. We combine these values using AND, OR, NOT, and XOR.

  • ANDBoth must be true
  • ORAt least one must be true
  • NOTFlips True ↔ False
  • XORExactly one must be true
1is_raining = True
2has_umbrella = False
3if is_raining and not has_umbrella:
4    print("You will get wet!")

Variables

Named containers for data

A variable is a labelled box in memory. You give it a name, put a value inside, and update it whenever you need.

2age          = 20       # int

4is_enrolled  = True     # bool
